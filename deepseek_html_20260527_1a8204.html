Installation

Install USME Core + VWAP Elite on ThinkorSwim, TradingView, or NinjaTrader (NinjaScript).

NinjaTrader (NinjaScript) – New Platform

1
Import NinjaScript Archive
In NinjaTrader 8, go to Tools → Import → NinjaScript Add-On. Select the USME_VWAP_Elite_Suite.zip file from your purchase email.
2
Enter License Key
After import, open a chart → Indicators → “USME VWAP Elite” → Inputs tab: paste your license key (same key works for all platforms).
3
Configure VWAP Anchors
Select from daily, weekly, session-based or custom anchored VWAP. The Elite suite auto-detects all standard deviation bands (1σ, 2σ, 3σ).
✓
NinjaTrader exclusive: Cumulative Delta + SOTD overlay — see volume footprint and USME signal simultaneously on the same chart.

ThinkorSwim & TradingView

Same as previous process: paste script from purchase email, enter license key. The USME VWAP Elite components are included as optional inputs (enable “Show VWAP Bands” to activate).

// USME VWAP ELITE SUITE — 01

VWAP Elite Suite

Volume-Weighted Average Price advanced toolkit: anchored VWAPs, multi-deviation bands, cumulative delta divergence + USME fusion.

Anchored VWAP & Dynamic Bands

The Elite suite introduces 7 VWAP modes: Session VWAP, Weekly, Monthly, Earnings-Anchor, Swing-High/Low Anchor, and Custom Timestamp. Each anchor auto-generates 3 standard deviation bands (1σ, 2σ, 3σ) which act as institutional support/resistance zones. USME uses these bands as extra confluence boosters: an ELITE signal that occurs exactly at the 2σ band receives a “VWAP Confluence” rating, raising score by +0.7.

VWAP + SOTD Fusion Filter

Enable Require_VWAP_Confluence to filter signals: only PRIME/ELITE signals that also align with VWAP band touch or rejection will be displayed. This drastically reduces whipsaws during low-volume periods.

// Example: VWAP elite logic (Pine/ThinkScript abstract)
if (signalTier == "ELITE" and priceTouchesVWAPBand(2)) {
    Alert("ELITE+VWAP: Ultra high conviction");
    confluenceBonus = 0.7;
}
📊
Backtest result (Feb–June 2025): ELITE + 2σ VWAP band touch → 78% win rate on NQ 15m, avg RR 2.8:1.

Cumulative Delta Divergence (NinjaTrader exclusive)

The NinjaScript edition embeds real-time bid/ask delta divergence. When price makes a higher high but cumulative delta makes a lower high (bearish divergence) while USME prints a bearish ELITE signal → ultra high-probability short entry.
